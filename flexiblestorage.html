Flexible Storage

November 3, 2010 / birch veneer, plastic

This prototype examines the potential for a modular, flat file storage system that hangs on a wall and is infinitely expandable. Birch veneer "blinds" run along wire tracks, creating pockets that conceal and reveal it's contents based on a user's interaction with it.
